# ML-Service CI/CD Project

## Стратегия деплоя
В проекте реализована стратегия **Rolling Update** с использованием Docker Compose. 
- **Преимущество**: Нулевое время простоя.
- **Механика**: Новые контейнеры запускаются по очереди, проверяются на работоспособность, после чего старые удаляются.

## Как запустить локально
1. Соберите образ: `docker build -t ml-app .`
2. Запустите деплой: `docker-compose -f ci.yml up -d`
